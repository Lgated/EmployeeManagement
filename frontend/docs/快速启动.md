# APJ员工管理系统 - 快速启动指南

## 🚀 快速开始

### 1. 安装依赖

```bash
cd frontend
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

### 3. 访问应用

打开浏览器访问：`http://localhost:3000`

## 📝 使用说明

### 首次使用

1. **注册账号**
   - 访问登录页面
   - 点击"还没有账号？立即注册"
   - 填写用户名、密码、邮箱（可选）
   - 点击"注册"

2. **登录系统**
   - 使用注册的账号登录
   - 登录成功后自动跳转到员工列表

3. **管理员工**
   - 查看员工列表
   - 新增员工
   - 编辑员工信息
   - 删除员工
   - 搜索员工

4. **查看统计**
   - 点击侧边栏"数据统计"
   - 查看各部门员工数量
   - 查看各部门平均薪资

## ⚠️ 注意事项

1. **确保后端服务已启动**
   - 后端服务运行在 `http://localhost:8080`
   - 如果后端未启动，前端无法正常工作

2. **数据库准备**
   - 确保PostgreSQL数据库已创建
   - 确保已执行建表SQL
   - 确保数据库连接配置正确

3. **CORS配置**
   - 确保后端已配置CORS
   - 允许前端域名访问
